## Controls

- **W-A-S-D** to move the first player.
- **T-F-G-H** to move the second player.
- **I-J-K-L** to move the third player.
- **8-4-5-6** to move the forth player.

### Debug:

- **F1** Change the scale.
- **F2** Change the scale.
- **F3** Change the scale.
- **F5** Desactivate the split screen.
